The HTML language consists of various tags, each serving a specific purpose. The following are examples of notable tags in HTML

  1. <html> </html>: defines the root of an HTML document.
  2. <head> </head>: contains the metadata about the HTML document.
  3. <title> </title>: sets the title of the HTML document.
  4. <body> </body>: encloses the content of the HTML document.
  5. <h1> to <h6>: defines heading levels, with <h1> being the highest.
  6. <p> </p>: represents a paragraph.
  7. <a> </a>: creates hyperlinks.
  8. <img>: Embeds images.
  9. <ul> </ul> <ol> </ol> <li> </li>: Establish unordered and ordered lists.
  10. <div> </div>: defines a division or a section in an HTML document.
  11. <dl> </dl>: defines a description list. The tag is used in conjunction with <dt> </dt> (defines terms/names) <dd> </dd>: (describes each term/name).
